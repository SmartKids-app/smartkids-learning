Module 02 • Keyboard Theory

Welcome to the Keyboard Kingdom ⌨️

Today we learn the important parts of the keyboard: home row keys, top row keys, bottom row keys, function keys, number keys, symbol keys, Enter, Backspace, Space Bar, Caps Lock, Shift key, Home, End, Page Up, Page Down, Ctrl and Alt keys.

Lesson 1 — Home Row Keys

The home row keys are A, S, D, F, J, K, L and ;. Your fingers rest here first. Great typing starts from the home row because your fingers can easily move up or down and return home again.

Lesson 2 — Top Row, Bottom Row & Number Keys

The top row contains Q to P and the number row above it contains 1 to 0. The bottom row contains Z to M. These help us type letters, numbers and complete words faster.

Lesson 3 — Function Keys & Alphanumeric Keys

Function keys are F1 to F12 at the top of the keyboard. They perform special computer commands. Alphanumeric keys are letters A–Z and numbers 0–9. They are used most often when typing.

Lesson 4 — Symbol Keys, Enter, Backspace & Space Bar

Symbol keys include @, #, $, %, &, and more. Enter starts a new line or confirms an action. Backspace deletes what is behind the cursor. Space Bar creates spaces between words.

Lesson 5 — Caps Lock, Shift, Ctrl & Alt

Caps Lock keeps letters in uppercase. Shift makes one capital letter or helps type symbols. Ctrl and Alt are command keys used together with other keys for shortcuts and special tasks.

Lesson 6 — Home, End, Page Up & Page Down

Home moves to the beginning. End moves to the last part. Page Up moves the screen upward and Page Down moves the screen downward. These help us navigate documents quickly.

Summary

A keyboard is more than letters. Every key has a purpose. When students understand keyboard theory first, practice becomes faster, smarter and easier. Learning the keyboard builds confidence for school, CBT exams and future digital careers.
