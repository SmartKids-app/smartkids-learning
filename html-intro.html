HTML Basics

Before websites become beautiful, they need structure. HTML is the skeleton of every webpage. Let’s understand the three main parts every webpage needs.

What Does HTML Mean? 🧱

HTML stands for Hyper Text Markup Language. It tells the browser what to show, where to show it, and how different parts of a webpage are arranged.

HHyper = connected links
TText = words and content
MMarkup = instructions for the browser
LLanguage = the system used to write it
🏠

Think of a House

HTML is the frame of the house before painting (CSS) and electricity (JavaScript).

The 3 Main Parts of a Webpage

🌍

<html>

This is the full house. It wraps everything on your webpage from start to finish.

🧠

<head>

This is the brain. It contains settings like title, styles, and important page information.

👀

<body>

This is what people see — headings, text, images, buttons, and everything visible.

Your First Full Structure 💻

<html>
  <head>
    <title>My First Page</title>
  </head>

  <body>
    <h1>Hello World!</h1>
  </body>
</html>

This is the starting point of almost every website on the internet.
