HTML Attributes ⚙️

Tags tell the browser what to show. Attributes give extra instructions about how it should work.

Think of attributes like giving more details. For example: not just “show image,” but “show THIS image.”

Example

<img src="photo.jpg">

src is the attribute. It tells the browser which image to show.

What Attributes Do 🔍

🖼️

Choose Images

Attributes help tell the browser which image file to display.

🔗

Create Links

They tell links where users should go after clicking.

🎨

Add Styling Help

Some attributes help with colors, design and page structure.

Common HTML Attributes You Must Know 🚀

src

Source

Used with images to show where the picture is coming from.

href

Hyperlink Reference

Used inside links to tell where the user should go.

alt

Alternative Text

Helps explain what an image is if it does not load.

class

Class Name

Helps organize and style parts of your webpage.

Real Attribute Examples 💻

<img src="school.jpg" alt="My School">
<a href="https://www.kidventor.com.ng">Visit Kidventor</a>
<div class="card">Welcome</div>

Notice how attributes give extra meaning to the tags.

Quick Practice ✍️

Which attribute helps an image know what picture to display?

href
src
class

Correct answer: src
