Module 1: Software

What is Software?

The invisible instructions that tell the hardware what to do!

1. The Simple Difference

If Hardware is the body of the computer, then Software is the mind.

Software is a set of instructions (code) that tells the hardware exactly how to perform a task. You cannot touch software, but you see it working every time you open an app or a game.

🏠

System Software

The "House" for your apps.

This is the Operating System (OS). It manages all the hardware. Examples: Windows, macOS, Android.

🎮

Applications (Apps)

Tools for specific tasks.

Programs you use to do work or have fun. Examples: Chrome, MS Word, WhatsApp, Minecraft.

📱 A Mobile Example

Your actual phone—the glass screen and battery—is Hardware. But Android or iOS is the System Software that makes it turn on. TikTok or Gmail are the Applications you use inside it!
